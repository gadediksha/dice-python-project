# 🎲 Dice Game

A simple command-line Dice Game built using Python where two players roll a dice and compete based on their total scores.

## 📖 Description
This project simulates a dice game between two players. Each player rolls a dice in turns, and their scores are added. At the end, the player with the highest score wins.

## 🚀 Features
- Two-player game
- Random dice roll (1 to 6)
- Automatic score calculation
- Winner declaration

## 🛠️ Tech Stack
- Python 3
- random module

## ▶️ How to Run

1. Save the file as `dice_game.py`
2. Open terminal / command prompt
3. Run:

   python dice_game.py

## 🎮 How to Play
- Player 1 presses Enter to roll the dice
- Player 2 presses Enter to roll the dice
- Scores are added automatically
- After all rounds, the winner is declared

## ⚠️ Note
Currently, the game runs 4 rounds:
for i in range(1, 5)

If you want only 3 rounds, change it to:
for i in range(1, 4)
